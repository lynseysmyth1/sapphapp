# Sapph - Dating App for Women

A dating app designed specifically for women.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Expo CLI (install globally with `npm install -g expo-cli`)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your device:
   - Scan the QR code with Expo Go app (iOS/Android)
   - Press `i` for iOS simulator
   - Press `a` for Android emulator
   - Press `w` for web browser

## Project Structure

- `App.js` - Main app entry point
- `components/SplashScreen.js` - Splash screen component
- `uploaded_images/` - Image assets

## Notes

- The app uses the Prompt font family for the logo
- Make sure to add the Prompt font files to `assets/fonts/` directory
- The splash screen displays for 3 seconds before showing the main app
